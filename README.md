# DjangoCourse

## project_1 
  First project to understand how to create a project and app using Django
  ### Important commands
    django-admin startproject [PROJECT_NAME] - creates a Django project
    python3 manage.py startapp [APP_NAME] - creates an app within a Django project

## monthly_challenges
  Second project to understand views and URLs in a Django app

  ### Important commands/functions
  #### Creating URLs
    urlpatterns - a list that holds paths to different routes
    path('[name of path]', views.function) - pass the name of the path and the view function that should be ran whenever this path is accessed.
